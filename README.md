# Imp005 EZ Eval Hardware Abstraction Layer

Use the IMP005_EZ_EVAL table to access the hardware pins, uart, spi, and i2c bus objects.

## IMP005_EZ_EVAL

* SENSOR_AND_IOH_I2C
* TEMP_HUMID_I2C_ADDR
* ACCEL_I2C_ADDR
* ACCEL_INT
* USB_EN
* USB_LOAD_FLAG
* ADC_SPI
* SHIELD_RESET
* LED_RGB_CLOCK
* LED_RGB_DATA
* IOL_UART
* IOL_2
* IOL_3
* IOL_4
* IOL_5
* IOL_6
* IOL_7
* IOH_8
* IOH_9
* IOH_10
* IOH_11
* IOH_12
* IOH_13
